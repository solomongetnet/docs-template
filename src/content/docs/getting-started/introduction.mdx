# Introduction

**Lumen** is a documentation framework built for developers who care about speed,
craft, and clarity. It pairs MDX content with a refined reading experience —
search, dark mode, accessible navigation, and zero configuration to get started.

## Why Lumen

Most documentation tools optimize for either authoring speed or reading
experience. Lumen optimizes for both. Write in MDX, ship a polished site.

- **Fast by default.** Pre-rendered, code-split, and aggressively cached.
- **Accessible.** Keyboard navigation, semantic landmarks, ARIA where it matters.
- **MDX-first.** Mix Markdown with React components without ceremony.
- **Searchable.** Built-in client-side search across every page.

> Lumen draws inspiration from the documentation sites at Stripe, Vercel,
> and Tailwind CSS — refined typography, generous whitespace, and content that
> always takes center stage.
